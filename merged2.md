## ðŸ§  Algorithm

Input two numbers

Read b

Inside the function:
3.1. Calculate sum = a + b
3.2. Calculate sub = a - b
3.3. Calculate mul = a * b
3.4. Return sum, sub, mul

Receive returned values

Assign sum, sub, mul = result(a, b)

Display the results

Print "Sum is <sum>, Sub is <sub>, & Multiply is <mul>"

Stop

## ðŸ§¾ Program

``````````````````````````````````````

    sub=a-b
    mul=a*b
``````````````````````````````````````
